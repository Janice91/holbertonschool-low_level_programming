# 0-isupper

## Description
Ce programme contient la fonction `_isupper` qui vérifie si un caractère est une lettre majuscule.  
Elle retourne 1 si c'est une majuscule (`A` à `Z`), 0 sinon.

## Fichiers
- `0-isupper.c` : Implémentation de la fonction `_isupper`.
- `main.h` : Contient le prototype de `_isupper`.
- `0-main.c` : Programme de test pour vérifier `_isupper`.
